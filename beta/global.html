    DisTubeOptions

    DisTube options.

    Type:
    • object
    Properties
    Name Type Attributes Default Description
    emitNewSongOnly boolean <optional>
    		 false

    @1.3.0. If true, DisTube#event:playSong is not emitted when looping a song or next song is the same as the previous one
    highWaterMark number <optional>
    		 1<<24

    @2.2.0 ytdl's highWaterMark option.
    leaveOnEmpty boolean <optional>
    		 true

    Whether or not leaving voice channel if channel is empty when finish the current song. (Avoid accident leaving)
    leaveOnFinish boolean <optional>
    		 false

    Whether or not leaving voice channel when the queue ends.
    leaveOnStop boolean <optional>
    		 true

    Whether or not leaving voice channel after using stop() function.
    searchSongs boolean <optional>
    		 false

    Whether or not searching for multiple songs to select manually, DisTube will play the first result if false
    youtubeCookie string <optional>
    		 null

    @2.4.0 YouTube cookies. How to get it: YTDL's Example
    youtubeIdentityToken string <optional>
    		 null

    @2.4.0 If not given, ytdl-core will try to find it. You can find this by going to a video's watch page, viewing the source, and searching for "ID_TOKEN".
    proxy string <optional>
    		 null

    @2.6.0 A https proxy.
